# NBA Regular Season Analysis (2025-26) 🏀

This project explores player performance in the 2025-26 NBA regular season using publicly available data from Basketball Reference.

## Project goal
The goal of this notebook is to clean the dataset and analyze basic player statistics such as:
- points per game
- assists per game
- rebounds per game
- blocks per game

## Data source
- Basketball Reference

## Main steps
1. Load per-game player statistics
2. Clean the dataset
3. Handle players who changed teams during the season
4. Create player rankings
5. Visualize top performers in selected categories

## Tools used
- Python
- pandas
- matplotlib
- Google Colab

## Future improvements
This project can be extended with:
- advanced statistics
- efficiency metrics
